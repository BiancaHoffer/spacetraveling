<h3>🏁 Start</h3> 

 ```
 # clonar repositório
 $ git clone https://github.com/BiancaHoffer/dt-money.git
 
 # acessar pasta
 $ cd spacetraveling
 
 # baixar dependências
 $ yarn add
 
 # iniciar projeto
 $ yarn dev
 
```
